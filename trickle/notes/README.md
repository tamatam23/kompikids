# Kompi Kids Landing Page

This is the landing page for Kompi Kids, a computer learning service for children.

## Structure
- **Navbar**: Navigation and CTA.
- **Hero**: Problem awareness and main value proposition.
- **Solutions**: Core benefits of the program.
- **Features**: Why choose Kompi Kids.
- **Programs**: Details on specific courses (Coding, Basic Computer, School Tasks, Design, Website).
- **Methods**: Learning modes (Online/Home Visit).
- **Testimonials**: Social proof.
- **Footer**: Contact info and social links.

## Tech Stack
- React 18
- TailwindCSS
- Lucide Icons
